# 高三资料领取招生系统说明书

## 以后怎么保存版本

以后代码保存、提交、上传 GitHub，默认由 Codex 处理。

你不用记终端命令。每次改完一版，只要说“保存一版”或“记得 git”，Codex 就负责检查当前改动、保存到 Git，并上传到 GitHub。

只有 GitHub 要输入密码或 token 时，需要你自己输入，因为账号密码不能交给别人。

## 系统定位

这是一个面向教培招生的最小运营单元原型：用一个资料领取 H5 页面，把家长从“看到资料”引导到“预约领取、留下联系方式、加老师微信、进入后续跟进”。

后台记录客户来源、页面停留、点击次数、分享关系和跟进状态，帮助老师判断谁更值得优先联系。

## 家长端流程

正式家长端入口：

```text
https://apply.xdianping.cn/zhaosheng
```

本地测试家长端入口：

```text
http://127.0.0.1:4173/zhaosheng
```

注意：`http://127.0.0.1:4173/` 是本地总览预览页，适合看界面，不建议用它测试 CRM 入库。

1. 打开活动链接，先看标题、人数、截止时间和资料预览。
2. 点击底部“立即领取资料”。
3. 底部出现领取方式，选择“到校自提”或“包邮到家”。
4. 到校自提填写：学生姓名、所在学校、联系电话。
5. 包邮到家填写：学生姓名、所在学校、联系电话、收件地址。
6. 所在学校支持输入后自动匹配合肥常见高中。
7. 提交后显示“领取预约已提交”，资料先为孩子预留。
8. 家长可以等老师联系，也可以长按二维码添加老师，发送“导数资料”领取题册和讲解视频。

## 分享键怎么用

页面右上角有“分享”按钮。

在手机微信、浏览器等支持系统分享的环境里，会优先调起系统分享面板；如果环境不支持，会自动复制分享文案和链接。

系统会记录分享点击次数。通过专属分享链接进入并提交领取的客户，会在后台分享关系里显示是谁带来的。

## 微信授权怎么用

当前已经加好微信授权骨架。没有配置公众号信息时，页面不会强制授权，家长端照常使用。

正式接公众号时，需要准备：

1. 公众号 AppID。
2. 公众号 AppSecret。
3. 公众号后台配置网页授权域名，例如 `apply.xdianping.cn`。
4. H5 正式部署在这个授权域名下面。

服务器启动时配置：

```bash
WECHAT_AUTH_ENABLED=1
WECHAT_APP_ID=你的AppID
WECHAT_APP_SECRET=你的AppSecret
python3 server.py
```

授权成功后，系统会记录：

- 微信 openid
- 微信昵称
- 微信头像
- 授权时间

后台 CRM 会显示“微信身份”。同一个微信在同一个活动里只能领取一次，重复提交不会新增第二个客户，也不会重复扣减名额；后台会在原客户记录里标记“重复提交已拦截”。

如果家长没有完成微信授权，系统会用手机号做兜底去重；所以手机号仍然需要家长填写。

本地测试可以在链接后加：

```text
?mock_wechat=1
```

这样会生成一个模拟微信身份，用来测试后台显示和去重逻辑。

## 后台模块

管理端入口：

```text
https://apply-admin.xdianping.cn/admin
```

首次部署后，服务器会自动生成后台密码文件：

```text
/www/apply-system/data/.admin-password
```

在阿里云服务器里查看密码：

```bash
cat /www/apply-system/data/.admin-password
```

如果以后要手动改密码，可以直接改这个文件，或在 systemd 里配置 `APPLY_ADMIN_PASSWORD`。修改后重启 `apply-system` 生效。

1. 活动管理：修改标题、副标题、名额、截止时间、老师微信、进群口令和分享文案。
2. 数据总览：查看访问、领取、诊断、试听、互动热度和今日跟进提醒。
3. 客户 CRM：查看客户信息、学生学校、来源、意向分、页面行为、分享贡献和跟进状态。
4. 渠道链接：创建不同传播渠道链接。
5. 分享关系：查看谁分享后带来了哪些客户。
6. 资料内容：添加文字、图片、视频、语音、PDF 等资料内容。
7. 表单配置：设置家长需要填写的信息。
8. 公开设置：控制前台人数、近期动态、头像、脱敏和名单展示规则。
9. 使用说明：查看系统说明、已实现功能和待实现清单。

## 多活动怎么用

后台支持多个资料领取活动并列发起。

在管理端顶部可以切换当前活动，也可以新建活动。新建活动时会复制当前页面内容，但访问、领取、CRM、分享关系会重新开始。

新建活动成功后，后台会弹出家长端链接，支持一键复制和打开预览。

家长端活动链接规则：

```text
https://apply.xdianping.cn/zhaosheng/活动后缀
```

例如：

```text
https://apply.xdianping.cn/zhaosheng/daoshu
https://apply.xdianping.cn/zhaosheng/yuanzhui
```

管理端“清空数据”只清空当前活动的访问、领取、CRM 和分享数据，不会删除页面标题、资料内容、表单配置和老师二维码。

不同活动之间的数据独立。同一个微信在 A 活动领过资料后，仍然可以在 B 活动重新领取。

## 异常领取怎么处理

如果最近领取动态里出现误填、捣乱或测试数据，可以在后台客户 CRM 处理。

1. 进入“客户 CRM”。
2. 点击对应客户。
3. 在右侧详情里选择：
   - “隐藏前台动态”：家长端不再显示这条近期领取，已领取数和剩余名额也会按有效记录重新计算；后台仍保留原始记录。
   - “恢复前台动态”：把隐藏记录重新显示到家长端。
   - “删除线索”：从 CRM 和近期领取里删除这条记录，同时清理渠道里这个客户的领取名字。

建议日常先用“隐藏前台动态”，只有明显测试或垃圾数据再删除。

## 数据安全规则

家长端公开接口只返回页面展示需要的数据，例如活动标题、资料预览、人数、近期领取和渠道数字。

手机号、openid、微信昵称、微信授权时间、访客访问明细、CRM 备注等完整客户数据，只在管理端域名读取。

## 当前需要注意

- 真实微信头像需要公众号配置完成后才能拿到；当前可用模拟微信身份测试。
- 当前二维码是占位图，上线前需要替换为真实老师或企业微信二维码。
- 文件上传还没有接正式对象存储，目前适合放小图、小视频或外部素材链接。
- 后台已接简单密码登录；正式多人协作时再升级为账号权限体系。
- 数据库、小程序版本还没有接入。
- H5 页面不能直接读取家长微信号，也不能直接调用微信收货地址；这些能力要走微信授权或小程序能力。
